# processor-emulator
> A modern C++ project exploring processor architecture through the development of a processor emulator.

---

## Why this project?

I've always been interested in how processors work, and I wanted a project that would let me explore that in a practical way while expanding my C++ knowledge.

Rather than building lots of unrelated practice programs, I'd rather spend time working on one larger project that grows over time. My hope is that it develops into something similar to the sort of software project I could imagine contributing to in industry.

The initial goal is to build a simple processor emulator from scratch, understanding each component as it's added. Once that's working, I'd like to experiment with more advanced ideas such as different cache architectures, measure how they affect performance, and gradually extend the project as my understanding grows.

Eventually I'd also like to use the project as a stepping stone into learning Verilog by implementing some of the hardware components separately.

---

## Development Approach

I'll be developing the project incrementally, taking the time to understand each component before adding the next.

Where appropriate I'll document interesting design decisions, alternative approaches and lessons learned as the project develops.

---

## Current Goals

- Develop a modular processor emulator in modern C++.
- Improve my understanding of processor architecture.
- Gain more experience using Git and GitHub throughout a long-term project.
- Build a project that can continue to grow as I learn.
